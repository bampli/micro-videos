# micro-videos
Video catalog admin backend

- dev tools
- app node.js + typescript
- category entity
- tests unit & integ
- uses cases & repo
- test integ

- other entities: genre, cast member, video

- nest.js api rest
- RabbitMQ & Encoder integ
- E2E tests
